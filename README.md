# Habit Tracker Analytics

Proyecto académico para el seguimiento de hábitos y análisis de progreso.

## Stack Tecnológico Final

El stack del proyecto queda definido y congelado de la siguiente manera:

- Frontend: React + TypeScript
- Backend: Python (Flask o FastAPI)
- Base de datos: SQL
- Análisis de datos: Pandas
- BI: Power BI
- Control de versiones: Git + GitHub

Después de esta definición, no se realizarán cambios de stack,
solo ajustes de implementación dentro de las tecnologías acordadas.
